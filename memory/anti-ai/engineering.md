# 工程类专属套话防治

| 套话 | 替代 |
|------|------|
| 精心组织科学施工 | 给施工部署 + 流水段划分 + 关键线路甘特图 |
| 确保工程质量 | 给质量目标 + 控制点 + 检验批 + 通病防治 |
| 高度重视安全生产 | 给安全责任制 + 专项方案 + 应急预案 + 演练频次 |
| 合理安排进度 | 给甘特图（关键线路）+ 进度保障措施 |
| 投入充足资源 | 给劳动力计划表 + 机械设备表 + 进场时间 |

## 要点
- 进度用甘特图标关键线路；工期不超招标最长工期
- 质量/安全措施落到控制点与专项方案，不喊口号
- 关键人员资质（建造师等级、证书）满足★要求
