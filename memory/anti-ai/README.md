# 反 AI 味与公文文风规则

投标书是正式公文，最忌"AI 味"与"空话套话"——评委一眼识别，主观分掉档。

本目录规则在 `init.py` 时按标书类型合成到项目 `.claude/knowledge/anti-ai.md`，由 writer 撰写前加载、reviewer 审查时核对：
- `common-rules.md` — 通用反 AI 味 + 公文文风规则（所有类型继承）
- `service.md` / `procurement.md` / `engineering.md` — 各类型专属套话防治

核心理念：**用可核验的事实、数据、流程、清单说话，删掉一切无信息量的形容词。**
