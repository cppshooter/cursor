# 硬件开发类技术方案 — 专属套话防治

| 套话 | 替代 |
|------|------|
| 硬件性能领先 | 给具体技术指标（设计值/实测值）+ 偏离表 |
| 完全满足技术要求 | 逐条技术参数偏离应答，写实际型号/参数 |
| 可靠性高 | 给可靠性设计（冗余/降额/MTBF）+ 试验验证 |
| 工艺精良 | 给工艺方案 + 关键工序质量控制点 |
| 兼容性强 | 给兼容范围 + 测试验证报告 |

## 要点
- 技术指标写设计值/实测值，不写"满足"
- ★项严禁负偏离；正偏离须服务于指标
- 可靠性/EMC/环境适应性须有设计论证与试验
- 只写技术，不写报价/资质/商务
