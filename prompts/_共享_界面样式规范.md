# 共享附录 B · QT 界面统一样式规范（功能模块界面设计强制遵守）

本附录适用于第 3 章「功能模块详细设计」下各子模块的「界面设计」小节。所有 HTML 原型必须遵守同一套视觉与结构规范，以便统一评审，并体现 Qt（Widgets/Fusion 风格）的视觉语言。

## 一、总体布局：标准 Qt 主窗口结构

每个功能模块界面都按下列 5 区结构组织（对应 `QMainWindow`）：

1. **顶部菜单栏**（`QMenuBar`）：文件 / 编辑 / 视图 / 工具 / 帮助 等通用菜单
2. **工具栏**（`QToolBar`）：本模块高频操作的图标按钮（≤8 个）
3. **左侧导航 / 树**（`QTreeView` 或 `QListWidget`）：模块内子功能或数据分类（可选）
4. **中央工作区**（`QWidget`）：表单 / 表格 / 图表占位（模块核心区）
5. **底部状态栏**（`QStatusBar`）：当前状态、连接状态、进度、时间

> 若某模块不需要左侧导航，可以省略，但其余 4 区必须保留。

## 二、统一配色与字体（Fusion 风格）

```css
/* 基础变量，所有界面 HTML 复制使用 */
:root{
  --qt-bg: #f0f0f0;            /* 窗口背景 */
  --qt-panel: #fafafa;         /* 面板/分组 */
  --qt-border: #b8b8b8;        /* 控件边框 */
  --qt-border-dark: #707070;   /* 焦点/分隔线 */
  --qt-text: #202020;          /* 主文本 */
  --qt-text-muted: #606060;    /* 次要文本 */
  --qt-primary: #2a82da;       /* 主操作色（Fusion 蓝） */
  --qt-primary-hover: #3a92ea;
  --qt-danger: #c62828;        /* 危险操作 */
  --qt-warning: #f9a825;       /* 警告/告警 */
  --qt-success: #2e7d32;       /* 成功 */
  --qt-row-alt: #e8e8e8;       /* 表格隔行 */
}
body{font-family:"Microsoft YaHei","Noto Sans CJK SC","PingFang SC",sans-serif;font-size:12px;color:var(--qt-text);background:var(--qt-bg);margin:0;}
```

## 三、统一控件类名（HTML 模拟 Qt 组件）

| Qt 组件 | HTML 类名 | 用途 |
|---|---|---|
| QMainWindow | `.qt-window` | 整体窗口容器 |
| QMenuBar / QMenu | `.qt-menubar` / `.qt-menu` | 顶部菜单 |
| QToolBar | `.qt-toolbar` | 工具栏容器 |
| QToolButton | `.qt-toolbtn` | 工具按钮 |
| QStatusBar | `.qt-statusbar` | 底部状态栏 |
| QPushButton | `.qt-btn` / `.qt-btn-primary` / `.qt-btn-danger` | 普通/主操作/危险按钮 |
| QGroupBox | `.qt-group` + `.qt-group-title` | 分组框 |
| QLabel | `.qt-label` | 标签 |
| QLineEdit | `.qt-input` | 单行文本输入 |
| QComboBox | `.qt-combo` | 下拉框 |
| QSpinBox | `.qt-spin` | 数字框 |
| QCheckBox / QRadioButton | `.qt-check` / `.qt-radio` | 勾选/单选 |
| QTableView | `.qt-table` | 数据表格 |
| QTreeView | `.qt-tree` | 树形结构 |
| QTabWidget | `.qt-tabs` + `.qt-tab` | 选项卡 |
| QProgressBar | `.qt-progress` | 进度条 |
| QTextEdit / 日志 | `.qt-textarea` / `.qt-log` | 多行文本 / 日志区 |
| QDialog | `.qt-dialog` | 弹窗（如确认对话框） |
| 状态指示灯 | `.qt-led` + `.qt-led-on/off/warn` | 硬件状态指示 |

## 五、统一交互约定（操作手册写作时同步引用）

- **菜单约定**（顶部菜单栏标题中括号字母为助记键）：
  - 文件(F)：新建任务、打开文件、保存、导出 Excel、退出
  - 编辑(E)：撤销、重做、查找、清除
  - 视图(V)：刷新、切换视图、显示/隐藏侧栏
  - 工具(T)：策略管理、硬件配置、自检、日志检索
  - 帮助(H)：用户手册、关于
- **快捷键约定**：Ctrl+N 新建、Ctrl+S 保存、Ctrl+F 查找、F5 刷新、Ctrl+L 清除日志、F1 帮助。
- **按钮顺序约定**：左→右 = 主操作（蓝色 `.qt-btn-primary`） → 次操作 → 取消/关闭。危险操作（清除、删除）使用 `.qt-btn-danger` 并弹出 `QDialog` 二次确认。
- **表单字段对齐**：标签右对齐，输入框统一宽度（短 120px、中 200px、长 320px）。
- **表格列宽**：操作列固定 100px，并位于最右。
- **状态指示**：硬件在线/离线/告警使用 `.qt-led-on/off/warn` 三色圆点。

## 六、生成时强制要求

1. 每个子模块的「界面设计」HTML **必须复制上述模板骨架**，再在 `qt-content` 区填入业务控件。
2. 业务区使用本附录所列 **`.qt-*` 类名**，不得引入 Bootstrap/Tailwind 等外部框架类名。
3. 操作步骤里描述的菜单 / 按钮 / 表单字段必须与界面 HTML 一一对应（同名同位置）。
4. 不使用图片、不使用外部字体/CDN，全部使用本附录的内联 CSS。
