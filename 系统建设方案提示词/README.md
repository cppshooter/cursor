# 系统建设方案提示词文件说明

本目录依据《系统需求.md》和《大纲.md》拆分生成系统建设方案各章节提示词。使用方式如下：

1. 先使用 `01-引言.md` 至 `09-售后服务与保障方案.md` 生成系统建设方案主体章节。
2. 编写第 3 章“功能模块详细设计”时，先使用 `03-功能模块详细设计.md` 生成章节总述，再分别使用 `03-功能模块详细设计/` 下各模块提示词生成模块详细设计。
3. 每个模块提示词均要求输出：功能模块描述、操作步骤、PlantUML 时序图、Java 类/算法设计、PlantUML 用例图、HTML 界面设计。
4. 所有提示词均以系统需求中的功能边界为准，允许拆分子功能，不允许扩大大模块范围。
