#  质量控制要求

1. 遵循的标准

本软件的研制生产过程严格遵循GJB438C-2021相关标准。

2. 设计约束

本设备的研制生产严格按照技术协议书/技术要求中的要求进行设计、投图、调试和生产工作。

3. 评审、测试、监造、监制、验收等要求

该采购外包品应进行设计评审，通知甲方参加所要求的评审，保留记录，问题进行闭环。

甲方根据项目计划，对本产品研制开发及生产过程进行监制、验收。

4. 对乙方保存的文件的检查要求

乙方应对本软件的研制生产过程中的工艺、记录、设计文件进行保存和归档，甲方有权对上述记录、文件进行检查。

5. 技术状态管理要求

乙方应按照GJB 438C-2021相关标准对软件的技术状态进行控制。

6. 关重件控制要求

本产品不是关重件。

7. 质保周期要求

乙方提供给甲方为期5年的质保维护，乙方应按照甲方要求派人参加系统联调、所检、外场、用户验收等，并共同解决期间出现的技术问题。

（1）合同签订10个工作日内，完成功能设计和界面设计；

（2）合同签订15个工作日内，完成总体设计和概要设计；

（3）合同签订60个工作日内，完成软件编码、单元测试；

（4）合同签订70个工作日内，完成软件的测试及试运行，包括集成测试、联调和试运行；

（5）合同签订90个工作日内，完成软件合格性测试及验收。

（6）2026年8月31日完成软件合格性测试及验收。

# 售后服务要求

乙方提供给甲方为期5年的维护，乙方应按照甲方要求派人参加系统联调，并共同解决联调中出现的技术问题。
