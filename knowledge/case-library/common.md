# 案例库 — 通用说明

本案例库为 retriever（检索员）提供"相似项目成功案例描述"与"技术参数基准"，供 writer 引用佐证。

> ⚠️ 重要：以下案例均为**脱敏的示范模板/行业基准**，非真实项目实绩。作者应导入本企业真实历史标书与项目案例替换/补充本库，retriever 引用时必须标注"已去敏"，writer 不得将示范案例当作本企业真实业绩填报（业绩须真实可举证，否则废标）。

## 案例条目格式

```markdown
### [案例编号] 案例标题
- **类型：** 服务 / 采购 / 工程 / 综合
- **规模：** 预算区间 / 体量
- **行业：** 政务 / 金融 / 制造 / ...
- **覆盖评分点方向：** 总体架构 / SLA / 进度 / 偏离 ...
- **成功案例描述（可引用，已去敏）：** ...
- **技术参数基准（可引用）：** ...
- **可引用片段：** ...
```
