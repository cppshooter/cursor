# 案例库 — 通用说明

本案例库为 retriever（检索员）提供"相似项目成功案例描述"与"技术参数基准"，供 writer 引用佐证。

> ⚠️ 重要：本系统仅编写**技术方案**，案例库只提供**技术层面**的成功案例与技术参数基准，不含商务/价格/资质业绩内容。以下均为**脱敏的示范模板/行业基准**，非真实项目实绩。作者应导入本企业真实历史技术方案替换/补充本库，retriever 引用时必须标注"已去敏"，writer 不得将示范案例当作真实业绩填报。

## 案例条目格式

```markdown
### [案例编号] 案例标题
- **类型：** 软件开发 / 硬件开发 / 系统集成 / 服务
- **技术覆盖：** 总体架构 / 选型 / 偏离 / 安全 / 可靠性 / 集成 ...
- **成功案例描述（可引用，已去敏）：** ...
- **技术参数基准（可引用）：** ...
- **可引用片段：** ...
```
