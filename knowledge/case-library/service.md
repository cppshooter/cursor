# 案例库 — 服务类（脱敏示范，仅技术）

### [SV-001] 某省级政务云运维技术服务项目
- **类型：** 服务
- **技术覆盖：** 服务技术体系、监控平台、分级技术响应、应急技术预案
- **成功案例描述（可引用，已去敏）：** 为某省级政务云提供运维技术服务，建立统一监控+自动化运维平台，分级技术响应，系统可用率 99.95%，重大故障平均恢复 28 分钟。
- **技术参数基准（可引用）：** 可用率≥99.9%；严重故障技术响应≤15min、恢复≤2h；监控覆盖率 100%。
- **可引用片段：** 采用监控告警+自动化处置闭环，故障诊断手册化，应急预案定期演练。

### [SV-002] 某行核心系统技术支持项目
- **类型：** 服务
- **技术覆盖：** 技术保障、变更技术流程、容灾切换
- **成功案例描述（可引用，已去敏）：** 为某行核心系统提供技术支持，建立严格变更技术窗口与回滚机制，年度零重大技术事故。
- **技术参数基准（可引用）：** 变更回滚预案 100% 覆盖；容灾切换演练≥2 次/年。
- **可引用片段：** 变更执行"测试—窗口实施—验证—回滚预案"技术闭环。
