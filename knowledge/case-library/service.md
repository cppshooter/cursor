# 案例库 — 服务类（脱敏示范）

### [S-001] 某省级政务云运维服务项目
- **类型：** 服务
- **规模：** 三年期、7×24 驻场+远程
- **行业：** 政务
- **覆盖评分点方向：** SLA、团队架构、运维制度、应急响应
- **成功案例描述（可引用，已去敏）：** 为某省级政务云提供 7×24 运维，建立三级服务台与升级机制，系统可用率达 99.95%，重大故障平均恢复时间 28 分钟，连续三年考核优秀。
- **技术参数基准（可引用）：** 可用率≥99.9%；严重故障响应≤15min、恢复≤2h；月度服务报告 + 季度持续改进。
- **可引用片段：** 采用 ITIL 流程，工单全生命周期管理，超时自动升级至项目经理与技术总监。

### [S-002] 某大型银行信息系统技术支持项目
- **类型：** 服务
- **规模：** 大型、混合驻场
- **行业：** 金融
- **覆盖评分点方向：** 团队资质、变更管理、应急预案
- **成功案例描述（可引用，已去敏）：** 为某大型银行提供核心系统技术支持，配置含 PMP/ITIL 认证的 30 人团队，建立严格变更窗口与回滚机制，年度零重大事故。
- **技术参数基准（可引用）：** 变更成功率≥99%；关键岗位持证上岗；7×24 值班 + 节假日加强值守。
- **可引用片段：** 变更管理执行"申请—评审—测试—窗口实施—验证—回滚预案"六步闭环。
