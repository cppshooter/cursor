### [SW-001] 某政务一体化平台开发项目
- **类型：** 软件开发
- **技术覆盖：** 总体架构、微服务、安全（等保三级）、高并发
- **成功案例描述（可引用，已去敏）：** 为某政务一体化平台采用微服务+容器化架构，支撑万级并发，通过等保三级，接口标准化对接多个委办局系统。
- **技术参数基准（可引用）：** 并发≥10000；接口响应≤500ms；等保三级；微服务+容器化+服务网格。
- **可引用片段：** 采用分层+微服务架构，统一 API 网关，数据库读写分离 + 分布式缓存保障性能。
