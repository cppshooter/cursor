# 案例库 — 系统集成类（脱敏示范，仅技术）

### [SI-001] 某市数据中心集成项目
- **类型：** 系统集成
- **技术覆盖：** 总体集成架构、网络拓扑、选型、容灾
- **成功案例描述（可引用，已去敏）：** 完成某市数据中心软硬件集成，双可用区容灾，全部技术指标正偏离响应，平滑对接现网系统。
- **技术参数基准（可引用）：** 双 AZ 容灾 RPO≤5min/RTO≤30min；核心设备冗余；兼容现网虚拟化平台。
- **可引用片段：** 总体集成架构统领选型，统一接口与协议保障互联互通，迁移割接含回退预案。

### [SI-002] 某企业信创集成替换项目
- **类型：** 系统集成
- **技术覆盖：** 国产化集成、兼容性、迁移割接
- **成功案例描述（可引用，已去敏）：** 完成某企业国产化软硬件整体集成替换，完全满足信创要求，迁移数据零丢失。
- **技术参数基准（可引用）：** 国产 CPU+OS 全栈；兼容主流外设与应用；迁移零丢失。
- **可引用片段：** 提供完整兼容性测试报告与割接回退方案，分批灰度降低风险。
