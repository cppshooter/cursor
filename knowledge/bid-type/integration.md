# 系统集成类技术方案撰写要点

适用：软硬件集成、平台集成、机房/网络/弱电集成等以"集成交付"为核心项目的**技术方案**编写。

> 编写依据只取招标文件的：**评分表-技术部分**、**实质性要求-技术产品部分**、**技术任务书 / 采购需求技术部分**。不涉及商务、报价、资质业绩等内容。

## 核心得分逻辑

技术评委关注：**总体集成架构是否合理、选型是否满足技术指标、互联互通与兼容性、迁移割接与联调能否平稳落地。**

## 强制完整性（缺项即失分/视为设计不完整）

系统集成类技术方案**必须**完整包含以下内容，逐项不可缺：

1. **需求理解与集成目标** —— 逐项理解 + 集成范围/边界 + 技术难点
2. **总体集成架构** —— 必须包含：**总体架构图、网络拓扑图、安全域划分、链路与冗余设计**
3. **软硬件选型与配置清单** —— 选型清单（型号/参数/数量）+ **技术参数偏离表**（★项无偏离/正偏离，无空项）
4. **集成方案与互联互通** —— 必须包含：**系统间接口/协议/数据对接方案 + 兼容性论证 + 标准符合性**
5. **迁移割接与联调** —— 必须包含：**数据迁移方案、割接步骤、回退预案、联调测试方案与验收指标**
6. **非功能与保障** —— 性能/可用性/容灾 + 安全设计（等保/边界防护/数据安全）

## 必备章节与侧重

### 1. 需求理解与集成目标
- 对技术任务书/采购需求技术部分的逐项理解
- 集成范围、边界、技术难点

### 2. 总体集成架构
- 总体技术架构、网络拓扑（引用 architecture/ 图）
- 安全域划分、链路与冗余设计

### 3. 软硬件选型与配置清单
- 选型清单（型号/参数/数量）对应技术要求
- 关键技术指标响应，形成**技术参数偏离表**（★项无偏离/正偏离）

### 4. 集成方案与互联互通
- 系统间接口/协议/数据对接方案
- 兼容性论证（与现网/既有系统）、标准符合性

### 5. 迁移割接与联调（技术侧）
- 数据迁移技术方案、割接步骤与回退预案
- 联调测试方案、验收技术指标

### 6. 非功能与保障（技术侧）
- 性能/可用性/容灾设计
- 安全设计（等保/边界防护/数据安全）

## 常见失分点

- 集成架构拼盘感、缺总体统领
- 选型与技术指标不匹配、偏离表空项
- 兼容性仅口头声明、无对接协议清单与测试
- 迁移割接无回退预案

## 实质性要求（技术产品部分）红线

- 实质性★技术指标负偏离 → 否决
- 强制标准/协议/国产化要求未响应 → 否决
- 必须的兼容性/测试承诺缺失 → 否决

- [ ] 迁移割接：迁移方案 + 割接步骤 + 回退预案 + 联调测试与验收指标
- [ ] 非功能与保障：性能/可用性/容灾 + 安全设计
